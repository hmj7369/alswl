# CCTV 실제 사진 넣는 곳

이 폴더에 아래 파일명으로 사진(JPG)을 넣으면, 해당 채널의 생성 장면 대신
**실제 사진**이 표시됩니다. 파일이 없으면 코드로 그린 장면이 그대로 보입니다.
사진을 넣은 뒤 브라우저를 새로고침하면 반영됩니다.

| 파일명            | 채널   | 장소              |
|-------------------|--------|-------------------|
| `ch-01.jpg`       | CH-01  | 중앙 로비          |
| `ch-02.jpg`       | CH-02  | 지하 주차장 B2      |
| `ch-03.jpg`       | CH-03  | 비상계단 3F         |
| `ch-05.jpg`       | CH-05  | 후문 통로          |
| `ch-06.jpg`       | CH-06  | 엘리베이터 홀       |
| `ch-07.jpg`       | CH-07  | 창고 구역(사건현장) |
| `ch-09.jpg`       | CH-09  | 정문 게이트         |
| `ch-10.jpg`       | CH-10  | 복도 A             |
| `ch-11.jpg`       | CH-11  | 복도 B             |
| `ch-12.jpg`       | CH-12  | 관리사무소          |

- CH-04, CH-08 은 `NO SIGNAL`(오프라인) 채널이라 사진을 넣지 않습니다.
- 화면 비율은 **16:9** 를 권장합니다(잘림 없이 채워짐).
- 사진에는 자동으로 CCTV 녹색 톤·스캔라인·비네팅이 입혀집니다.
- 인물/단서 위치가 사진과 어긋나면 `script.js` 의 CHANNELS 에서
  해당 채널 `hotspots` 의 x/y(%) 만 조정하면 됩니다.
